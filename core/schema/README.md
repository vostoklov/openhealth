# Core Schema

Zod schemas and TypeScript types for the Health OS data model.

## Key Types

- `HealthEvent` — the fundamental unit of health data
- `HealthCategory` — categories of health data (sleep, activity, vital, etc.)
- `HealthConnector` — the interface every connector must implement
- `Hypothesis` — community health experiment definition
- `AnonymizedResult` — anonymized experiment result
