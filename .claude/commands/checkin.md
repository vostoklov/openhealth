Use the **health-agent** skill. Ask one plain question: "How are you feeling
right now?" Offer simple options (great / good / okay / low) but accept free
text. Save it as a note via the existing intake (a telegram-intake style
envelope or a manual note) so it joins the timeline. Scan the text for any
red-flag symptom; if present, stop and route to professional care. Keep it warm
