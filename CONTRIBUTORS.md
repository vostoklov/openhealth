# Contributors

OpenHealth is built by a community. This file credits the people whose ideas,
patterns, reference norms, and code shaped the project — including ideas adopted
from their own Health-OS builds.

How crediting works (see [RFC-002](rfcs/002-community-contribution-model.md) and the
`contributor-harvest` skill):

- We adopt **ideas, patterns, reference norms, and insight templates** — never
  anyone's personal health data (RFC-002 Stream 3 stays on the owner's machine).
- A person is credited here **only with their explicit consent**, alongside the
  public source they shared.
- Code contributions are also credited via `Co-Authored-By` trailers and PR history.
