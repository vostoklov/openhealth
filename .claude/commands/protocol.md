Use the **health-agent** skill. Help the person design a simple n-of-1 test for
a hypothesis from $ARGUMENTS (e.g. "does cutting late coffee improve my HRV?").
Following docs/methodology/evidence-and-trust.md:
- baseline first (1–2 weeks), change one thing, repeat the on/off switch, allow
  washout, watch confounders.
Write it down as a short plan (a ValidationPrompt-style note) and remind them a
personal pattern stays a weak signal (C2) until it survives one repeated switch.
No diagnosis, no prescription.
