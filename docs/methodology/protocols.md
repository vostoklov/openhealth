# Протоколы (n-of-1 эксперименты)
> algo_version: n/a (модуль protocols, шаблоны-константы) · источник данных: движок · редактируемость: параметры в коде

## Что это

Мост от «возможно, есть проблема» (инсайт) к «вот одно изменение и точный критерий успеха» (протокол). Это единственный путь поднять личный паттерн с C2 (слабый сигнал) к C3 (гипотеза): минимальный single-subject эксперимент с baseline-фазой и заранее заявленным числовым критерием.

## Формула / алгоритм

**Схемы:**

- **ABAB** — baseline (A) → вмешательство (B) → снова A → снова B. Повторное переключение отделяет эффект от совпадения. Используется для большинства протоколов (7 дней на фазу; для недельных паттернов — 14).
- **AB** — одно переключение, когда возврат к baseline нежелателен или бессмыслен (красная серия recovery, фиксация времени подъёма).

**Правила дизайна (зашиты в код):**

1. Одно вмешательство на протокол — иначе результат неинтерпретируем.
2. Числовой критерий успеха заявлен до старта (no moving goalposts), например: «среднее recovery в фазах B выше A на >= 5 пунктов».
3. `confidence_cap = C2`, пока протокол не завершён: само наличие эксперимента доверия не добавляет.
4. На каждом протоколе note по безопасности; для красной серии — усиленный («сначала к врачу, потом эксперимент»).

**Построение** (`build_protocols`): из инсайтов — по шаблону на каждый вид (sleep_debt, hrv_downtrend, rhr_uptrend, recovery_red_streak, strain_recovery_mismatch, weekend_pattern, sleep_consistency); из корреляций — ABAB-проверка для C2+ привычек (C1 отбрасывается). Ранжирование: severity инсайта, затем confidence; максимум 3 активных предложения. Для HRV-протокола вмешательство таргетируется: если среди негативных корреляций есть классический подавитель HRV (алкоголь/экраны/поздно/кофеин — `_HRV_TRIGGER_HINTS`), протокол предлагает убрать именно его.

**Критерии успеха шаблонов (примеры):** ранний отбой → recovery в B выше A на >= 5 пунктов; HRV-протокол → 7д среднее HRV в B выше базового на >= 8%; RHR → 7д пульс покоя в B в пределах 2 уд/мин от baseline; из корреляции → планка `max(3, round(|impact|))` пунктов (не гоняться за шумом).

## Источники и доверие

- n-of-1 / ABAB — стандартная методология single-subject экспериментов.
- Канон доверия — `openhealth/evidence.py`: завершённый протокол с повторным переключением может поднять паттерн до C3, не выше.

## Известные ограничения

- Слепления и плацебо не контролируются — ты знаешь, в какой фазе живёшь.
- Сезонность/болезни могут совпасть с фазой; открытые вопросы протокола напоминают это проверить.
- Движок предлагает и оценивает протоколы, но не принуждает: исполнение и честность фаз — на человеке.
